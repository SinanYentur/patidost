# MVP Definition (Phase 1)

## Core User Flow:
1.  **Authentication:** User can register and log in using email/password.
2.  **Profile Management:** User can view and update their basic profile information.
3.  **Pet Discovery:** User can swipe through a list of pet profiles.
4.  **Interaction:** User can "like" or "pass" on a pet.
5.  **Matching:** When a mutual "like" occurs, a match is created (notification is Phase 2).

## Must-Have Features (Epics):
*   **Epic 1: Kimlik Doğrulama ve Profil Yönetimi**
*   **Epic 2: Evcil Hayvan Keşif ve Eşleştirme**
*   **Epic 3: Ana Navigasyon ve Temel Akış**

## Launch Criteria:
*   No critical (app-crashing) bugs.
*   Core user flow is fully functional.
*   Basic analytics and crash reporting are integrated.
*   A placeholder Privacy Policy is available.
